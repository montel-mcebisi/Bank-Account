Bank Account Management System

A simple Java application that demonstrates the fundamentals of object-oriented programming by implementing a basic bank account system. The program allows users to deposit money, withdraw money, and check their account balance while preventing overdrafts.

Features
Create a bank account with an initial balance of 0.0.
Deposit funds into the account.
Withdraw funds from the account.
Prevent withdrawals that exceed the available balance.
Check the current account balance.
Technologies Used
Java
Object-Oriented Programming (OOP)
